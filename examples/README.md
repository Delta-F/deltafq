# 示例代码

## 示例列表

### ma_strategy.py - 双均线策略
基础策略示例，展示数据获取、指标计算、策略构建和回测流程。

```bash
python ma_strategy.py
```

### macd_strategy.py - MACD策略
基于MACD指标的策略示例。

```bash
python macd_strategy.py
```

### optimization_example.py - 参数优化
使用网格搜索优化策略参数。

```bash
python optimization_example.py
```

## 安装要求

```bash
pip install deltafq
pip install matplotlib  # 可选，用于绘图
```
