## [0.1.0] - 2025-10-11

### 新增
- 初始版本发布
- 数据模块：支持基础数据获取
- 技术指标：SMA, EMA, MACD, RSI, BOLL
- 策略框架：Strategy基类
- 回测引擎：基础向量化回测
- 风险管理：仓位管理、风险指标计算
- 绩效分析：收益率、夏普比率等指标
- 参数优化：网格搜索优化器
- 实盘交易接口：Broker抽象基类
- 工具函数：时间处理等
- 完整的单元测试
- 示例代码和文档
